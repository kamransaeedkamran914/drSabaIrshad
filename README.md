# Dr. Saba Irshad Website

This repository contains the source code for the personal and professional website of **Prof. Dr. Saba Irshad**.

## Website Structure
The website is located in the `website/` directory and includes:
- **Home**: Introduction and Hero section.
- **About**: Biography and Education.
- **Research**: Research interests and focus areas.
- **Publications**: Selected publications.
- **Contact**: Contact information.

## Tech Stack
- HTML5
- CSS3 (Material Design 3 inspired)
- JavaScript

## Deployment
This site is intended to be hosted on GitHub Pages.
