# Prof. Dr. Saba Irshad - Personal Website

This repository contains the source code for the professional website of **Prof. Dr. Saba Irshad**, Professor at the School of Biochemistry and Biotechnology, University of the Punjab.

## Website Structure

- **index.html**: Main landing page.
- **css/**: Stylesheets (Material Design 3 inspired).
- **js/**: JavaScript functionality.
- **assets/**: Images and other assets.

## How to Host on GitHub Pages

1. Create a new repository on GitHub.
2. Push this code to the repository.
3. Go to **Settings** > **Pages** and select the `main` branch as the source.
